# Login background video

Place your video file here so the login page can use it:

- **Filename:** `Kidsplaying.mp4`
- **Full path:** `apps/web/public/Kidsplaying.mp4`

Then the app will serve it at `/Kidsplaying.mp4` and the login page will show it as the background (muted, no sound).

If the file is missing, the login page shows a gradient fallback instead.
